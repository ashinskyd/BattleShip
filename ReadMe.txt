The file "tartarus.ini" located in the ____ folder contains the settings used for our genetic program. The default settings provided have generally good results but these values may be changed.

The external code came from the gpjpp/Tartarus project and the modifications we made can be found in Grid.java, Tartarus.java, TartGene.java, and our new file battleBoard.java. The Grid file includes our new terminals and functions, the tartarus file has changes to allow our program to run with our battleboard file, TartGene is modified to handle our new terminals and functions, and battleBoard now replaces most of the functionality of the Grid file.
